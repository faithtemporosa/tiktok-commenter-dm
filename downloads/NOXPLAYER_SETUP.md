## Installation (10 minutes)

### Step 1: Download NoxPlayer

1. Go to: https://www.bignox.com/
2. Click **"Download for Mac"**
3. Wait for download (~500 MB)

### Step 2: Install

1. Open the downloaded `.dmg` file
2. Drag **NoxPlayer** to **Applications** folder
3. Open **Applications** → **NoxPlayer**
4. Click **"Open"** if Mac asks for permission

### Step 3: First Launch

1. NoxPlayer will start (takes 2-3 minutes first time)
2. You'll see an Android home screen
3. Wait for "Ready" message

✓ Installation complete!

## Setup TikTok Account 1 (15 minutes)

### Step 1: Install TikTok

1. In NoxPlayer, click **Google Play Store**
2. Sign in with any Google account (or create one)
3. Search **"TikTok"**
4. Click **"Install"**
5. Wait for installation (~2 minutes)

### Step 2: Login to TikTok

1. Open **TikTok** app
2. Sign in with your **Account 1**
3. Verify it's working (browse For You page)

### Step 3: Record Automation Macro

This is the magic - you'll record actions once, then replay them!

**Click the Macro Recorder:**
1. On the right side of NoxPlayer, click **"Macro"** button (keyboard icon)
2. Click **"Create a new macro"**
3. Name it: **"View Target Account"**
4. Click **"Record"** (red circle button)

**Now perform these actions (NoxPlayer is recording):**

1. In TikTok, tap **Search** (magnifying glass)
2. Tap search box
3. Type: **"charlidamelio"** (or your target account)
4. Tap on **@charlidamelio** profile
5. Tap **first video**
6. Wait **15 seconds** (just wait, it's recording the pause)
7. Swipe **up** (to next video)
8. Wait **15 seconds**
9. Swipe **up**
10. Wait **15 seconds**
11. Swipe **up**
12. Repeat steps 10-11 about **10 times** (total ~10 videos)

**Stop Recording:**
1. Click **"Stop"** button in Macro window
2. Click **"Save"**

**Set to Loop:**
1. In Macro window, click **"Settings"**
2. Set **"Loop times"** to **3** (will watch 30 videos total)
3. Click **"Save"**

✓ Macro recorded!

### Step 4: Test the Macro

1. Go back to TikTok home screen
2. In Macro window, click **"Play"** button
3. Watch as NoxPlayer **automatically**:
   - Searches the account
   - Opens videos
   - Watches and swipes
   - Repeats!

**If something goes wrong:**
- Click **"Stop"** in Macro window
- You're not stuck! You have full control

## Setup Accounts 2 & 3 (10 minutes each)

### Create Multiple Instances

NoxPlayer can run 3 Android phones at once!

**Step 1: Open Multi-Instance Manager**
1. Click **"Multi-Drive"** button (top toolbar)
2. Click **"Add emulator"**
3. Select **"New emulator - Android 9"**
4. Click **"Create"**
5. Wait 5 minutes for creation
6. Repeat for 3rd instance

**Step 2: Setup Each Instance**
1. Start Instance 2
2. Install TikTok
3. Login with **Account 2**
4. Record macro (same steps as above)
5. Repeat for Instance 3 with **Account 3**

## Daily Usage (5 Minutes Setup, Runs Automatically)

### Morning: Run All 3 Accounts

**Step 1: Start All Instances**
1. Open **Multi-Instance Manager**
2. Click **"Start All"**
3. Wait for all 3 to load (~3 minutes)

**Step 2: Run Macros**
1. In **Instance 1**, click **Macro** → **Play**
2. In **Instance 2**, click **Macro** → **Play**
3. In **Instance 3**, click **Macro** → **Play**

**Step 3: Walk Away**
- All 3 accounts are now auto-viewing!
- Takes ~20 minutes total
- You can watch on screen to verify
- Click **"Stop"** in any instance if needed

**Step 4: Close When Done**
- After 20 minutes, macros stop automatically
- Close NoxPlayer
- Views registered!

## Troubleshooting

### NoxPlayer is Slow
- Settings → Performance
- Set CPU: 4 cores
- Set RAM: 4GB
- Enable "High Performance"

### TikTok Not Loading
- Make sure you have internet connection in NoxPlayer
- Settings → Network → Bridge mode

### Macro Not Working
- Re-record with slower actions
- Add longer wait times (20-30 seconds)
- Make taps more precise

### Can't See What's Happening
- This is the benefit of NoxPlayer - you CAN see!
- Just look at the screen
- Everything is visual
